Quem sou eu?

Apaixonada tanto pelo tradicional livro quanto às tecnologias, me formei em Biblioteconomia e Gestão da Informação e, com o meu interesse em Banco de Dados, decidi cursar Sistemas de Informação na Pontifícia Universidade Católica de Minas Gerais (PUC-MG). Estou em busca de oportunidades para desenvolver e obter novos conhecimentos técnicos, especialmente relacionado a Desenvolvimento Web e Análise de Dados.
Entrei para tecnologia com o intuito de ser Back-End, mas minha admiração em na área de Design também me fez olhar para o Front-End. Então, minha meta profissional é me tornar uma Dev Full-Stack.
